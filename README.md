# Zero Knowledge Proofs 研究项目

零知识证明（Zero-Knowledge Proofs）学习与实验。

## 目录

- `docs/` — 学习笔记、论文摘要
- `src/` — 实验代码
- `examples/` — 示例与 demo

## 技术栈

- Python 3.11+
- 相关库: `py_ecc`, `galois`, `sympy` 等（按需添加）

## 研究方向

- [ ] ZKP 基础理论（交互式证明、Sigma 协议）
- [ ] zk-SNARKs 原理与实现
- [ ] zk-STARKs 原理与实现
- [ ] Groth16 / PLONK 等证明系统
- [ ] 实际应用场景探索

## 快速开始

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
